# 语法
```
<style>

        h1 {

            color: red;

        }

  

        p {

            font-size: 50px;

        }

  

        a {

            color: black;

        }

    </style>

</head>

  

<body>

    <h1>给h1标签设置了颜色为红色</h1>

    <p>给p标签设置了字体大小为50像素</p>

    <a href="#">给a标签设置了颜色为黑色</a>

  

</body>
```
