# 语法
```
<style>

        .nav {

            color: red;

        }

    </style>

</head>

  

<body>

    <div class="top1 nav">盒子名称为top1，调用了选择器，所以为红色</div>

    <div class="top2 nav">盒子名称为top2，调用了选择器，所以为红色</div>

    <div class="top3 nav">盒子名称为top3，调用了选择器，所以为红色</div>

    <div class="top4 nav">盒子名称为top4，调用了选择器，所以为红色</div>
    
</body>
```
