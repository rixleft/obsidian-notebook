# 语法
```
<style>

        <style>* {

            margin: 0;

            padding: 0;

        }

  

        .top1 {

            float: left;

            width: 200px;

            height: 100px;

            background: yellow;

        }

  

        .top2 {

            float: left;

            width: 200px;

            height: 100px;

            background: purple;

            padding: 10px;

            border: 10px solid #000;

            box-sizing: border-box;

        }

    </style>

  

</head>

  

<body>

    <div class="top1">标准盒，只有宽和高，无内外边距和边框</div>

    <div class="top2">怪异盒，不需要自己计算，将其转换即可</div>

  

</body>
```
