# 语法
以下三种实现的效果相同，都是使`<h1>`标题标签颜色为红色。
```css
<style>
        h1 {
            color: red;
        }
        h1 {
            color: rgb(255, 0, 0);
        }
        h1 {
            color: #ff0000;
        }
</style>
```
