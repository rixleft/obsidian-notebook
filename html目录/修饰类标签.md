```html
<!DOCTYPE html> 
<html lang="en"> 
<head> 
<meta charset="UTF-8"> 
<meta http-equiv="X-UA-Compatible" content="IE=edge"> 
<meta name="viewport" content="width=device-width, initial-scale=1.0"> 
<title>修饰类标签</title> 
</head>

	<b>加粗</b>
	<strong>加粗</strong>
	<i>倾斜</i>
	<em>倾斜</em>
	<u>下划线</u>
	<ins>下划线</ins>
	<del>删除线</del>
	<s>删除线</s>
	<sup>上角标</sup>
	<sub>下角标</sub>
	水平线是单标签<hr>
	强制换行标签<br>是单标签
</body> 
</html>
```
