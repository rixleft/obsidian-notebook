```
在 无序列表的 li 标签里填写有序列表的type属性，可改变单个 li 标签的序号，如：
<ul>

        <li type="a">无序列表</li>

        <li type="A">无序列表</li>

        <li type="1">无序列表</li>

        <li type="i">无序列表</li>

        <li type="I">无序列表</li>

</ul>
	
```

## 3.自定义列表
```
<dl>
         <dt>问题</dt>
         <dd>答案</dd>
         <dd>答案</dd>
</dl>

注意：一个自定义列表里有一个dt 可以同时存在多个dd。
     默认情况下dd是存在一定的缩进效果
```
