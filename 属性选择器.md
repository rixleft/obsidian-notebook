# 属性选择器
| 属性                  | 描述 |
| --------------------- | ---- |
| `input[value]`        | 只要元素存在此属性就可以生效     |
| `input[value="aaa"]`hu  |      |
| `input[value|="aaa"]` |      |
| `input[value^="aaa"]` |      |
| `input[value$="aaa"]` |      |
| `input[value*="aaa"]`                      |      |
