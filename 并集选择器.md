# 语法
类型一
```
<style>

        .paragraph1,

        .box {

            color: red;

        }

    </style>

</head>

  

<body>

    <p class="paragraph1">颜色</p>

    <div class="box">颜色</div>

  

</body>
```
类型二
```
<style>

        p,

        div {

            color: red;

        }

    </style>

</head>

  

<body>

    <p class="paragraph1">颜色</p>

    <div class="paragraph2">颜色</div>

  

</body>
```
