# 语法
==类型一：并集选择器==
```
<style>

        p,

        div,

        h5,

        b,

        em {

            color: red;

        }

    </style>

</head>

  

<body>

    <h5 id="top" class="nav">标题标签</h5>

    <p id="top" class="nav">段落标签</p>

    <b id="top" class="nav">加粗标签</b>

    <div id="top" class="nav">倾斜标签</div>

    <em id="top" class="nav">倾斜标签</em>

  

</body>
```
==类型二：多个==即为交集选择器
```
<style>

        #top {

            color: red;

        }

    </style>

</head>

  

<body>

    <h5 id="top" class="nav">标题标签</h5>

    <p id="top" class="nav">段落标签</p>

    <b id="top" class="nav">加粗标签</b>

    <div id="top" class="nav">倾斜标签</div>

    <em id="top" class="nav">倾斜标签</em>

  

</body>
```
==类型三：类选择器==
```
<style>

        .nav {

            color: red;

        }

    </style>

</head>

  

<body>

    <h5 id="top" class="nav">标题标签</h5>

    <p id="top" class="nav">段落标签</p>

    <b id="top" class="nav">加粗标签</b>

    <div id="top" class="nav">倾斜标签</div>

    <em id="top" class="nav">倾斜标签</em>

  

</body>
```
