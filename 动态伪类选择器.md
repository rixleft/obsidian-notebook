## 动态伪类选择器
| 属性     | 描述                               |
| -------- | ---------------------------------- |
| :link    | 初始状态，未访问过的链接           |
| :hover   | 鼠标悬停，当用户鼠标放在链接上时   |
| :active  | 鼠标点击未抬起，链接被点击的那一刻 |
| :visited | 鼠标点击过后，用户已访问过的链接   |
| :focus   | 获取焦点，表单input标签常用        | 
 
生效一次就不能生效了，因为代码的生效顺序有问题
==**正确的顺序：link  visited  hover active **==
