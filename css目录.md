# css
使用css可以弥补html标签本身的不足，可以减少代码冗余，可以减小网页体积，可以减少网络带宽占用，可以提高用户体验，有利于SEO，对网页的重构改版有很好的支持。

## 选择器

| 通配符选择器     | 0.0.0.0  |
| 标签选择器       | 0.0.0.1  |
| 类选择器         | 0.0.1.0  |
| ID选择器         | 0.1.0.0  |
| 交集选择器       | 权重相加 |
| 并集选择器       |          |
| 关系选择器       |          |
| 伪元素选择器     |          |
| 属性选择器       |          |
| 伪类选择器       | 0.0.1.0  |
| 行内样式style="" | 1.0.0.0  |
| !important       | 无穷大         |
