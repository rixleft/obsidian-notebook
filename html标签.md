| 标签           | 描述                 |
|:-------------- |:-------------------- |
| `<!DOCTYPE>`   | 定义文档类型         |
| `<html>`       | 定义一个HTML文档     |
| `<title>`      | 为网页取一个标题     |
| `<body>`       | 定义文档的主体       |
| `<h1>to<h6>`   | 定义HTML标题标签     |
| `<p>`          | 定义一个段落标签     |
| `<br>`         | 强制换行             |
| `<hr>`         | 水平线（单标签）     |
| `<!--...-->`   | 注释，快捷键为Ctrl+/ |
| `<b>,<strong>` | 加粗标签             |
| `<i>,<em>`     | 倾斜标签             |
| `<u>,<ins>`    | 下划线标签           |
|                |                      |
