# 语法
如果后代未设置样式，则会继承自己的样式，不论是儿子还是孙子。
```
<style>

        div {

            color: red;

        }

    </style>

</head>

  

<body>

  

    <div>

        布局

        <div>

            布局<span>我是div里面的span</span>标签

        </div>

        <p>

            段落

            <span>

                我是div里p标签的里面的span标签

            </span>

            标签

        </p>

        标签

    </div>

</body>
```
